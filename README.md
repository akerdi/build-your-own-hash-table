# Build-your-own-hashtable

Build your own hashtable

## Description Hashtable

[//]: # (Hashtable is a linking data structure, constructed by setting data in a strategy: calculating index of a key, fill with pointer of key and value.)

Hashtable 跟大多数语言的Map 方法功能相同，通过key 找到对应的value。快速 `插入`/`查找`及自由伸缩大小，使得Hashtable 被频繁使用。

本文是对[原文《write-a-hash-table》](https://github.com/jamesroutley/write-a-hash-table)的翻译和简化，使得更快学习完。

以以下顺序完成学习进程:

1. 先完成附加知识学习，如素数作用/素数查找方法
2. 完成基础hashtable方法
3. 解决冲突Collision
4. 伸缩表
